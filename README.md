# Camp-FE
Camp-FE

### 실행방법 (프론트)

1)
```bash
git clone https://github.com/Camp-MobileProgramming/Camp-FE.git
cd Camp-FE
git switch dev
cd FE
```
2)
```bash
npm install
```
3)
```bash
npm run dev
```

### ngrok 실행
1) ngrok 설치

2) ngrok 실행
```bash
ngrok http 8081
```
3) forwarding 주소로 접속
